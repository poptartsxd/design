    • MoviePass Redesign Concept

    Case Study


    MoviePass is a service that allows movie lovers to view as many movies as they want each month, booking showtimes on a mobile app. However, there are some hiccups in the mobile experience, requiring some refinement in the interaction.

    I worked with one other designer to redesign the interaction and visual aesthetics of the app.


    Redesign

    The App

    Problem: Create a seamless interaction from booking a showtime to watching the movie
    Users: MoviePass Subscribers
    Role: UX/UI Designer, Visual Designer
    Details: Partner Project, 48 hours

    Note: this is a redesign concept done by 2 HCI students. We do not own the logos/images/movie posters used in the wireframes as these were only placed there to demonstrate a concept.



    New Bookings Tab

    Users will be able to view, edit, cancel their current bookings. They will be notified when the allotted 30 minutes to book is up.



    Improved Navigation

    Users will be able to navigate the app via the native iOS tab bar. We redesigned the information architecture so only relevant and useful information is presented.



    Improved Search

    Default search will be to user's current location. Users will also be able to switch between List and Map view to find theaters.






    Process

    Understanding user needs

    The most common tasks done on the app are: 1) finding a theater, 2) finding a movie to watch, and 3) booking a showtime. We asked a few MoviePass users to conduct these tasks (and a few others) on the current app.






    Current user flow

    We identified areas for improvement by drawing out the current user flow.

    Low-Fidelity User Flows and Wireframes

    Based on user feedback and the weaknesses we identified in the current app, we created low fidelity wireframes of a new design.

    Thoughts

    This was a fun project as we both are movie lovers. We learned how to use formative testing in order to drive our designs, getting consistent feedback from users. Because we are also users of MoviePass, we had to be careful to not let our bias affect our designs. To alleviate this, we continuously asked for feedback from other users when designing.
